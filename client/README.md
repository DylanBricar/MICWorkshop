# Reactapp

### Init npm package

npm init -y

### Install create react app

npm i -g create-react-app

### Create a new racat app

create-react-app <your-app-name>

### Install socket io client

npm i socket.io-client

### Install react routers

npm i react-router-dom

### Add boostrap

npm i bootstrap

### Install moment

npm i moment

### Install sass

npm i sass

### Install axios

npm i axios

### Run your app

npm start
